# airfoil-tools
Airfoil design and generation tools with NACA4 support, curved profiles, and export to PTS and DXF.

![Manta](images/manta.jpg)

## Running the application

On Windows:
- Double-click `airfoil-tools.bat`

Or from terminal:

```bash
python airfoil_tools.py
```

Use only the canonical launchers:
- `airfoil-tools.bat` on Windows
- `python airfoil_tools.py` from terminal
